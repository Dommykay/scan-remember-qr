# scan-remember-qr
Making this so I don't have to carry a paper ticket for the bus.

The idea is that I can scan the QR code on my ticket, and it will be remembered locally on the device.
A button can be pressed which opens the camera and allows scanning of another QR code, at which point the old one is forgotten and the new one replaces it.
I could just use my camera, but there is so much extra bloat and steps that i'd rather just make this.
